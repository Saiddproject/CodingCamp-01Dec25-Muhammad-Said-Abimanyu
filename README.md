Megapa bisa menjadi guest? Variabel userName menjadi "guest" karena kode Anda menggunakan operator OR (||) sebagai nilai default: prompt(...) || "guest". Ketika pengguna mengklik OK tanpa memasukkan teks, fungsi prompt() mengembalikan string kosong (""). Dalam JavaScript, string kosong dianggap sebagai nilai falsy. Oleh karena itu, operator || mengabaikan hasil prompt() yang falsy tersebut dan langsung mengambil nilai kedua yang truthy ("guest"), yang kemudian ditetapkan sebagai nama pengguna.
